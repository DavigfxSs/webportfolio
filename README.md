## 💻 **Como acessar**
1. Clone o repositório:  
   ```bash
   git clone https://github.com/DavigfxSs/webportfolio.git
   ```
2. Instale as dependências:  
   ```bash
   npm install
   ```
3. Inicie o servidor local:  
   ```bash
   npm start
   ```
4. Acesse no navegador:  
   [http://localhost:3000](http://localhost:3000)
